===========================================
FILEZILLA TIPS:
===========================================

1. Select files in FileZilla (not folders)
2. Right-click → Upload
3. If file fails:
   - Check if file is open in another program
   - Try renaming file (remove special chars)
   - Upload one at a time

4. After upload, set permissions:
   - Right-click file → File permissions
   - PHP files: 644
   - Folders: 755
