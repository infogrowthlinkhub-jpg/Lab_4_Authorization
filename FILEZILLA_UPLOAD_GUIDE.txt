===========================================
FILEZILLA SETTINGS:
===========================================

1. TRANSFER SETTINGS:
   - File Type: Binary (for images, if any)
   - For PHP/JS/CSS: Auto or ASCII

2. FILE PERMISSIONS (after upload):
   - PHP files: 644
   - Directories: 755
   - db.php: 600 (more secure, but 644 works too)

3. UPLOAD PATH:
   - Remote directory: /public_html/
   - Upload all files directly to public_html (not in subfolders)
